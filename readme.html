Важные моменты

1. На торце корпуса (со стороны тачпада, справа от SD слота) есть два переключателя:

  • Переключатель видеокарты (со светидиодом) - выключает и включает дискретную видеокарту NVIDIA. При горящем светидиоде - видеокарта включена, при выключенном - выключена. При выключении видеокарты она не используется нигде, не отображается в диспетчере задач и диспетчере устройств. Вместо нее в это время используется интегрированная видеокарта Intel.
  • Переключатель Wi-Fi (еще чуть правее переключателя видеокарты) - выключает и включает модуль Wi-Fi. При выключенном модуле Wi-Fi перестанет работать. Bluetooth-модуль в данной модели отсутствует, однако я приложу USB адаптер Bluetooth в комплекте.

2. Для лучшей производительности рекомендуется использовать только дискретную видеокарту NVIDIA, особенно при игре или работе с графикой.

3. Сверху корпуса (где кнопка питания) также есть сенсорная панель регулировки громкости звука. Однако, эта панель с регулированием громкости не работает (шлейф не прикреплен). Если будете просить замену динамиков (подробнее в разделе "Экран и корпус"), попросите также присоединить шлейф этой сенсорной панели и шлейф тачпада (если он вам нужен, т.к. шлейф тачпада очень короткий). Про тачпад подробнее в "Недостатках".

4. В Windows 10 информация о скорости зарядки аккамулятора может отображаться неправильно, но сам уровень заряда отображается корректно.

5. Для данного ноутбука крайне рекомендуется использовать Windows 10. На более старых версиях Windows многие программы уже не поддерживаются, и оптимизация 10 лучшая для этого ноутбука. Windows 11 сюда просто официально не установить, а для Linux отсутсвуют драйвера.
